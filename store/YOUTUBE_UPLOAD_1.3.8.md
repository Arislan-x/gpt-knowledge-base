## 一、产品详情（中文）

### 产品名称

GPT 知识库

### 一句话介绍

跨平台实时备份当前 AI 会话，将有价值的对话归档到本地，并整理为可追溯、可导出的 RAG 知识资料。

### 产品详情

GPT 知识库是一款开源的 Chrome Manifest V3 扩展，用于汇聚分散在不同 AI 平台中的对话。它会备份浏览器中当前打开的会话，并在统一工作站中按平台整理、阅读、搜索、筛选和导出，让重要对话不再散落在各个平台。

开启“归档到本地”后，浏览器备份和导入内容可以同步写入用户主动授权的本地文件夹。即使扩展被移除、浏览器缓存丢失或重新安装，只要重新选择以前的归档文件夹，即可读取已有归档。

主要功能：

- 实时备份当前打开的 AI 会话，不主动抓取历史会话列表；
- 支持 ChatGPT、Claude、Grok、DeepSeek、Kimi、Gemini、Perplexity、Poe、千问、豆包、腾讯元宝、文心一言、智谱清言和 Hugging Face Chat；
- 按平台自动分组，在一个工作站中浏览和搜索会话；
- 支持 Markdown、表格、代码块、LaTeX 公式、思考折叠和问题跳转；
- 可将浏览器备份同步到用户授权的本地文件夹；
- 可重新选择旧归档文件夹，在缓存丢失或重新安装后恢复会话；
- 支持导入 JSON、Markdown、ZIP、备份文件夹和 ChatGPT 官方导出文件夹；
- 导入内容默认写入已授权的本地归档，同一会话重复出现时保留最新版本；
- 支持单个、多个或全部会话导出为 JSON、Markdown、HTML 或 ZIP；
- 会话数据默认保存在用户设备上，不上传到开发者服务器；
- 项目源代码依据 GPL-3.0 公开。

备份范围仅包括可识别的对话文本和内容结构，不会下载或保存图片、视频、音频、上传附件或模型生成文件本体。请使用原平台提供的下载功能单独保存这些资源。

GPT 知识库当前完成了跨平台采集、会话浏览、来源合并、选择和结构化导出，为后续的分块、索引、检索和本地 RAG 工作流提供数据基础。

## 二、Product Details (English)

### Product name

GPT Knowledge Base

### One-line description

Back up open AI conversations across platforms, archive valuable chats locally, and export them as traceable source material for RAG workflows.

### Product details

GPT Knowledge Base is an open-source Chrome Manifest V3 extension for collecting conversations scattered across AI platforms. It backs up the conversation currently open in the browser and brings supported chats into one workstation for platform-based organization, reading, search, filtering, selection, and export.

When Archive locally is enabled, browser backups and imported conversations can be synchronized to a local folder explicitly authorized by the user. If the extension is removed, browser storage is lost, or the extension is reinstalled, selecting the previous archive folder restores access to the archived conversations.

Key capabilities:

- Back up the currently open AI conversation in real time without proactively fetching conversation history lists;
- Support ChatGPT, Claude, Grok, DeepSeek, Kimi, Gemini, Perplexity, Poe, Qwen, Doubao, Yuanbao, Yiyan, Qingyan, and Hugging Face Chat;
- Group conversations by platform and browse or search them in one workstation;
- Render Markdown, tables, code blocks, LaTeX formulas, collapsible thinking sections, and question navigation;
- Synchronize browser backups to a user-authorized local folder;
- Reopen an existing archive folder after browser storage loss or reinstallation;
- Import JSON, Markdown, ZIP archives, backup folders, and official ChatGPT export folders;
- Write imports to the authorized local archive by default and keep the latest version of duplicated conversations;
- Export one, selected, or all conversations as JSON, Markdown, HTML, or ZIP;
- Keep conversation data on the user's device by default without uploading it to developer servers;
- Provide public source code under GPL-3.0.

The backup scope is limited to recognizable conversation text and content structure. Images, video, audio, uploaded attachments, and model-generated binary files are not downloaded or preserved. Save those resources separately using the original platform.

The current release provides the data foundation for cross-platform capture, conversation browsing, source merging, selection, and structured export. Future RAG capabilities can build chunking, indexing, retrieval, and local knowledge-base workflows on top of this structured conversation data.
