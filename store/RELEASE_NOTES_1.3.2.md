# GPT 知识库 v1.3.2

本版本增强了外部备份导入和工作站批量整理能力。

## 新增与改进

- 导入入口可选择 ZIP 备份包或本地备份文件夹。
- 支持多选会话，并将所选会话导出为 JSON、Markdown 或 HTML。
- 所选会话可合并为单个文件，也可分别打包为 ZIP。
- 支持批量删除所选的浏览器备份；外部文件夹原文件不会被删除。
- 导入弹窗和中英文 README 明确说明：外部 ZIP / 文件夹仅供当前工作站页面临时读取，不会复制进浏览器永久储存。
- 优化窄窗口工作站、平台图标选中环和右侧用户问题导航的显示。

## 手动安装

1. 下载 `gpt-knowledge-base-1.3.2.zip` 及对应的 `.sha256` 文件。
2. 将 ZIP 完整解压到一个固定文件夹。
3. 在 Chrome 打开 `chrome://extensions` 并启用开发者模式。
4. 点击“加载已解压的扩展程序”，选择解压目录。
5. 打开受支持的 AI 会话页面并刷新一次。

## 隐私和授权

实时捕获的会话永久保存在浏览器本地。外部 ZIP 和文件夹只在用户主动选择后临时读取，不会上传到开发者服务器，也不会修改电脑中的原文件。
