# GPT 知识库 v1.3.1

首个 GitHub Release。本文件保留历史版本说明；项目自 v1.3.3 起采用 GPL-3.0，当前许可见根目录 `LICENSE`。

## 核心能力

- 实时备份当前打开的跨平台 AI 会话，数据默认保存在浏览器本地。
- 在统一工作站中按平台浏览、整理并标记数据来源。
- 支持 Markdown、表格、代码块、LaTeX、思考折叠和用户问题导航。
- 支持单会话及批量 JSON、Markdown、HTML 导出，以及 ZIP 打包。
- 支持 ChatGPT、Claude、Grok、DeepSeek、Kimi、Gemini、Perplexity、Poe、千问、豆包、腾讯元宝、文心一言、智谱清言和 Hugging Face Chat。

## 手动安装

1. 下载 `gpt-knowledge-base-1.3.1.zip` 及对应的 `.sha256` 文件。
2. 将 ZIP 完整解压到一个固定文件夹。
3. 在 Chrome 打开 `chrome://extensions` 并启用开发者模式。
4. 点击“加载已解压的扩展程序”，选择解压目录。
5. 打开受支持的 AI 会话页面并刷新一次。
