# GPT 知识库 v1.3.7

`v1.3.7` 修复弹窗无法识别当前页面已有备份的问题，并保持精简后的权限范围不变。

## 修复内容

- 修复所有受支持 AI 平台的弹窗可能错误显示“当前页面暂无备份”的问题。
- 弹窗现在通过已有内容脚本读取当前页面地址，不需要重新申请 `activeTab` 或 `tabs` 权限。
- 当前页面备份匹配新增平台会话 ID、`localId`、来源 URL 会话 ID 和平台域名别名支持。
- 兼容 URL 查询参数变化，以及 ChatGPT 的 `chatgpt.com` 与 `chat.openai.com` 域名差异。

## 数据与权限

- 已有本地备份无需迁移，不会被删除或重建。
- 权限继续保持为 `storage`、`unlimitedStorage` 和清单中声明的受支持网站访问范围。
- 对话数据仍只保存在用户本机浏览器，不上传开发者服务器。

## 更新方式

1. 下载 `gpt-knowledge-base-1.3.7.zip`；`.sha256` 文件仅用于可选完整性校验。
2. 将 ZIP 完整解压到固定文件夹。
3. 在 Chrome 的扩展管理页重新加载扩展。
4. 刷新已打开的 AI 会话页面，使新版内容脚本生效。

Chrome Web Store 上传包为同一个 `gpt-knowledge-base-1.3.7.zip`。
