# GPT 知识库隐私政策

生效日期：2026 年 7 月 16 日

GPT 知识库是一款将当前打开的 AI 对话保存到用户本机、并支持浏览和导出的 Chrome 扩展。本政策说明扩展如何处理数据。

## 处理的数据

扩展仅在用户打开受支持的 AI 对话页面时处理以下数据：

- 当前打开会话的标题、消息正文、角色、来源网址和更新时间；
- 为完成本地分组、去重和导出所需的技术元数据；
- 用户主动选择导入的本地 JSON、Markdown 或兼容备份文件。

这些信息可能包含网站内容、浏览活动、个人通信和用户生成内容。扩展不会主动获取用户的历史会话列表，也不会读取未打开的会话。

## 使用目的

上述数据仅用于扩展的单一目的：在用户设备上实时备份、浏览、整理和导出当前 AI 会话，以便构建可追溯的本地知识资料和 RAG 数据源。

## 储存与传输

- 数据保存在 Chrome 的本地扩展储存或用户主动选择的本地文件中；
- 扩展不将会话内容、网址或使用数据上传到开发者服务器或第三方分析服务；
- 扩展不包含广告、跟踪器或遥测；
- 扩展不出售、出租或分享用户数据，也不允许开发者或其他人员读取用户保存的会话。

点击平台图标或“打开来源”只会在新标签页中打开对应的官方网页，不会把本地备份发送给该网站。

## 权限说明

- `activeTab`：识别用户当前打开的受支持会话，并在用户点击刷新时重新采集该页面；
- `storage`：在浏览器本地保存设置、会话索引和备份内容；
- `unlimitedStorage`：避免长会话和较大本地知识库受默认扩展储存额度限制；
- 受支持网站的内容脚本范围：仅在清单中列出的 AI 对话网站读取当前页面的会话 DOM，以实现产品声明的实时备份功能。

## 用户控制

用户可以随时在弹窗中暂停实时备份、删除单个会话、清空本地知识库，或卸载扩展。卸载扩展会由 Chrome 清除该扩展的本地储存；用户自行导出的文件不受影响。

## Chrome Web Store Limited Use

本扩展对用户数据的使用仅限于提供和改进其面向用户的单一功能。用户数据不会用于广告、信用评估、数据经纪或其他无关目的，不会转移给第三方，也不会由人员读取。本扩展的数据处理遵守 Chrome Web Store User Data Policy（包括 Limited Use 要求）。
