Privacy Policy

隐私政策

生效日期：2026 年 7 月 16 日

处理范围

扩展仅在用户打开受支持的 AI 对话页面时处理当前会话的标题、消息正文、角色、来源网址、更新时间，以及本地分组、去重和导出所需的技术元数据。扩展不会主动获取历史会话列表，也不会读取未打开的会话。

扩展只备份可识别的文本和内容结构，不下载或保存图片、视频、音频、上传附件、模型生成文件及其二进制内容。

储存与传输

  • 实时捕获的会话保存在 Chrome 本地扩展储存中。
  • 外部 ZIP 和文件夹只在用户主动选择后临时读取，不会写入浏览器永久储存。
  • 扩展不上传会话内容、网址或使用数据，不包含广告、跟踪器或遥测。
  • 用户数据不会被出售、出租、分享或供开发者读取。

权限与用户控制

activeTab 用于识别和刷新当前受支持会话；storage 用于保存本地设置和备份；unlimitedStorage 用于避免长会话受默认配额限制。用户可以随时暂停备份、删除会话、清空知识库或卸载扩展。

Chrome Web Store Limited Use

用户数据只用于提供和改进扩展声明的单一功能，不用于广告、信用评估、数据经纪或无关用途，也不会转移给第三方或由人员读取。

联系方式

问题、数据请求或安全报告可通过 GitHub Issues 提交。

English

Privacy Policy

Effective date: July 16, 2026

Data handled

When a supported AI conversation is open, the extension processes the open conversation title, message text, roles, source URL, update time, and technical metadata required for local grouping, deduplication, and export. It does not proactively fetch conversation history or read conversations that are not open.

Only recognizable text and content structure are backed up. Images, video, audio, uploaded attachments, model-generated files, and binary contents are not downloaded or preserved.

Storage and transfer

  • Live-captured conversations are stored in Chrome extension local storage.
  • External ZIP and folder imports are read only for the current workstation session.
  • Conversation content, URLs, and usage data are not uploaded. The extension contains no advertising, trackers, or telemetry.
  • User data is not sold, rented, shared, or made available for the developer to read.

Permissions and user control

activeTab identifies and refreshes the open supported conversation; storage stores local settings and backups; unlimitedStorage prevents long conversations from being constrained by the default quota. Users can pause backup, delete conversations, clear the knowledge base, or uninstall the extension at any time.

Chrome Web Store Limited Use

User data is used only to provide and improve the extension's single user-facing purpose. It is not used for advertising, credit assessment, data brokerage, or unrelated purposes, and is not transferred to third parties or read by humans.

Contact

Questions, data requests, and security reports can be submitted through GitHub Issues.
